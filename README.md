# Introduction to Deep Learning
Assignment for Introduction to Deep Learning

Student(s) name: Hyunmo Kang

Title: Dog vs Cats Image Classification

Summary (200 words): This report focuses on comparing different deep learning approaches for image classification using the Kaggle Dogs vs Cats dataset. The goal of this project is to classify images of dogs and cats using both traditional Convolutional Neural Networks and modern Transformer-based architectures, specifically the Vision Transformer, and to compare their performance and characteristics.

First, the report will introduce the fundamental concepts of image classification and explain how CNN models process image data through convolution and pooling operations. A basic CNN model will then be implemented using the PyTorch framework to perform the Dogs vs Cats classification task. The model’s training process and performance metrics such as accuracy and loss will be analyzed.

Next, the project will explore the Vision Transformer, a recent Transformer-based model that has gained significant attention in computer vision research. Unlike CNNs, ViT processes images as sequences of patches and uses self-attention mechanisms to understand global relationships within images. The ViT model will also be trained and evaluated on the same dataset.

Finally, the report will compare the strengths and weaknesses of CNN and ViT models in terms of accuracy, training efficiency, computational cost, and overall image classification performance.
